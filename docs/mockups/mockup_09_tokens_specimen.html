Chart vocabulary6 allowed types · everything else is forbidden

Percentile bar

Baseball Savant style. Single metric vs cohort distribution.

Trajectory spark

160×40 inline sparkline. No axis — direction only.

Bump chart

Two-column rank shifts (cohort vs cohort, week vs week).

Annotated line

NYT-style. Line + one or two annotation lines marking inflections.

Small multiples

Tufte / Bloomberg grid for comparing across many entities.

Heatmap

Density / time-of-day / week-by-week intensity.

Forbidden chart types

  • Pie charts — always. Percentile bars communicate the same data more honestly.
  • Vertical bar charts — use percentile bars instead; they encode position within distribution.
  • Radar / spider charts — except for the player fingerprint module (PLAYER_PAGE_WORLD_CLASS_BRIEF documents the exception).
