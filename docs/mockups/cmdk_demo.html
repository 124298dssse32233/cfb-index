Command-K overlay demo

Sprint v5-11.5 foundation. Press ⌘K (macOS) or Ctrl+K (Win/Linux), or click the trigger button below.

What's indexed in this demo

18 synthetic items spanning all 7 kinds — profiles, teams, conferences, editions, mailbag (empty in this demo), players, and methodology pages. The real build emits ~9,300 items.

Try these searches

  • alab — fuzzy match on Alabama (profile, tier 1)
  • QB — finds players by subtitle (position)
  • sec — finds conference + SEC teams via subtitle
  • cita — finds methodology pages
  • empty input — browse mode, grouped by kind
