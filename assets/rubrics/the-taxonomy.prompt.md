# The Taxonomy — Rubric Icon 04

## One-line brief
A simple three-node tree diagram. Icon for the taxonomy rubric.

## Visual family
Rubric icon — mono-on-bone ligne-claire, icon-chip family (no amber).

## Full prompt used
```
Single standalone 1:1 square illustration — not a grid, not a contact sheet, not panels. One image only.

Match the uploaded totem_master.png style exactly: flat ligne-claire on bone paper #F3EEE4, uniform 2px warm-ink contour, flat fills only, no hatching, no shadow, no gradients. Clean mid-century European cartooning discipline.

MONO-ON-BONE: no amber, no second color. Warm-ink line and warm-ink flat fills only on bone-paper ground. This is an icon-chip, not a totem.

SUBJECT: a minimal tree diagram with one filled circle at top and two filled circles below, connected by short straight lines in a Y-branching. Warm-ink circles, warm-ink connectors. No lettering.

COMPOSITION: centered on the paper with ~20% margin on all sides. Icon-chip proportions — the subject reads clearly at small sizes.
```

## Reference images
- `assets/masters/totem_master.png` (family reference only — this is a mono-on-bone icon, not an amber totem)

## Model + settings
- ChatGPT Pro, Art Director Custom GPT (gpt-image-1 backend), 1:1, single image.

## Selection rationale
The Y-branch is the most universally recognized 'classification' or 'taxonomy' figure. Three nodes is the smallest structure that shows branching — anything more would compete with icon-chip legibility.

## Rejection notes
None — single-image workflow, committed on first generation.
