# Generational Hope — Archetype Totem 08

## One-line brief
A shooting star frozen mid-arc above a goalpost. Visual shorthand for the program whose whole season is riding on one kid.

## Visual family
Ligne-claire totem, warm-ink + amber on bone paper. Matches locked master plate.

## Full prompt used
```
Single standalone 1:1 square illustration — not a grid, not a contact sheet, not panels. One image only.

Match the uploaded totem_master.png style exactly: flat ligne-claire on bone paper #F3EEE4, uniform 2px warm-ink contour, flat fills only, no hatching, no shadow, no gradients. Clean mid-century European cartooning discipline.

SUBJECT: a five-pointed shooting star frozen mid-arc with a short tail trail, suspended above a single goalpost silhouette. The goalpost rises from the bottom edge, the star hangs above it. No humans, no team logos, no stadium detail.

AMBER: the star itself is amber #E0A300. The goalpost and trail are warm ink. Exactly one amber detail — the star.

COMPOSITION: goalpost grounded bottom-center, star off-center upper-right, negative space upper-left, ~20% paper margin.
```

## Reference images
- `assets/masters/totem_master.png`

## Model + settings
- ChatGPT Pro, Art Director Custom GPT (gpt-image-1 backend), 1:1, single image.

## Selection rationale
Clean two-element read. The tail trail reads as three swept curves from lower-left to upper-right, which naturally directs the eye toward the amber star — motion as metaphor for the prospect's ascent. Goalpost silhouette centered-bottom keeps the ground plane honest. Minimalism here is a feature — this is a lot of negative space, which reinforces "the one kid" loneliness of the archetype.

## Rejection notes
None — single-image workflow, committed on first generation.
